*Data Modeling*
What is a table?
-A table is something that holds data with x or y.

What is an Entity?
-A segment/ an object within a table.

What is an Attribute?
-A certain characteristic of an entity.

What are the Data types in a DB?
-boolean, text, Date

What is a Primary key?
-Unique identifier for each item in row, usually computer generated.

What is a Schema?
-A picture that says what the data in a table is going to be.
